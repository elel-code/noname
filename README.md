# TCM Combination Optimization Template

This project provides a "minimal runnable" entry point for quick self-checking in dependency-free/read-only environments, while retaining the original optimization demo based on `pymoo` and `pyswarms` as an optional mode.

For detailed usage instructions, please refer to [USAGE.md](USAGE.md).
For an in-depth explanation of the ACI metric and API documentation, please refer to [references.md](references.md).
